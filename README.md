## 🏃‍♂️ 실행 방법

[이 레포지토리](https://github.com/walking-sunset/assignment-api)의 서버를 로컬에서 실행한 다음 본 레포지토리를 `clone`하여 다음 명령어를 입력해 주세요.

```bash
yarn
yarn dev
```

위 명령어 입력 후 `http://localhost:5173/`에 접속해 주세요.
